# 1\. 출력하기

1. 다음 단어를 출력하시오. `Hello`
2. 다음 문장을 출력하시오. `Hello World`
3. 다음 문단을 출력하시오. (줄바꿈)
   ```java
   Hello
   World
   ```
4. 다음 단어를 출력하시오.
   ```java
   'Hello'
   ```
5. 다음 단어를 출력하시오.
   ```java
   "!@#$%^&*()"
   ```
6. 다음 경로를 출력하시오.
   ```java
   "C:\Download\hello.cpp"
   ```
7. 다음 특수문자를 출력하시오.
   ```java
    ┌┬┐
    ├┼┤
    └┴┘
   ```
